"Wake on LAN": The key facts

"Wake on LAN" (WOL) means that a computer can be switched on remotely via its network card - even via the Internet.

  • Windows must be set correctly for WOL to work (see below).
  • Your network card must support "Wake on LAN".
  • In addition, you usually have to set up certain accesses - for example via FTP or VNC (Virtual Network Computing).
  • You also need a fixed IP address (or Domain / DynDNS).
  • For the network wake-up service to work, it sometimes has to be activated in the BIOS first.

Set "Wake on LAN" in Windows

"Wake on LAN" is activated/deactivated in the Windows Device Manager in the network adapter properties. Under Windows 10, "Wake on LAN" may not work properly. This is mostly due to wrong drivers and wrong Windows settings.

Install the correct driver for your network adapter.
Press the key combination [Windows] + [R] to open the Run dialog. Type devmgmt.msc to open the device manager.
Double click on your adapter in the "Network Adapters" category.
Switch to the "Power Management" tab and check the box "Device can activate the computer from sleep mode".
If you only use Magic Packets, you can also check the box "Only Magic Packet can activate computers from hibernation".

Also (for Realtek adapters) look in the tab "Advanced" and find the entry "Wake up with Magic Packet" in the list.
On the right, the drop-down menu should show the value "Enabled". If not, this changes to this value. Click on "OK".

Disables the Quick Start in Windows 10, which often causes problems in this context. Since Windows 8, Quick Start puts the PC into a special power saving mode when shutting down. However, Intel chips do not react then any more on WOL.

If "Wake on LAN" still doesn't work for you, the BIOS will boot. Depending on your computer, there are also settings for WOL that have to be activated first.
